# Employee Management Demo

A Spring Boot 3 backend application for managing employee records using Spring Web, Spring Data JPA, H2 in-memory database, Lombok, and Springdoc OpenAPI Swagger UI.

## Installation

1. Clone the repository or download the project files.
2. Open a terminal in the project root directory.
3. Run the application using Maven:

```bash
./mvnw spring-boot:run
```

4. Access the API at `http://localhost:8080/api/v1/employees`.
5. Open the H2 console at `http://localhost:8080/h2-console`.
6. View Swagger UI at `http://localhost:8080/swagger-ui.html`.
